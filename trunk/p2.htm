Technical Specifications

Compatible with IEEE 802.3, 802.3u, 802.3z, 802.3ab (pending)
Canary Fast Ethernet CFT series WDM converters comply with all applicable IEEE 802.3u specifications. Canary WDM converters are designed to comply with all the following safety, emissions and susceptibility specifications: UL 1950, CSA-C22.2 No. 950-93, CE Mark EN60950 and EN55022, and FCC Class B.

Diagnostic LED Indicators

System Power:   (PWR) LEDs- lit steady when power is received (one per PWR supply.)
TX Link:   (TX) LED- lit when the 100base-TX connection at the RJ-45 port is correct and has a good link (one per UTP port.)
FX Link:   (FX) LED- lit when the fiber optic cables are properly installed and has a good link (one per Active conversion port.)
WDM Port Link(s):   (WD) LED- lit when good F/O link is received
(one WDM link indicator per Active TX or FX conversion port.)
Link Fault Signaling:   (LF) LED- lit if either the fiber optic or twisted pair link has failed (only for Active TX or FX conversion port.)
Full Duplex:   (FD) LED- lit when, Active 100 Mbs TX/FX port is set to full-duplex, full-duplex traffic is present.

Environmental
Operation temp:   0° to 49° C
Storage temp:   -10° to 66° C
Relative humidity:   5% to 95% non-condensing
